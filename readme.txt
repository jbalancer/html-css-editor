var настройки = {
	уведомления: "включить",
	превью: "динамическое",
	функциональность: {
		очистка: "да",
		копирование: "да",
		сохранение: "да"
	}
};
/*-----------------------------------------*/
Если в настройках:
	- (уведомления: "включить"/"отключить"), то при каждом нажатии кнопок и срабатывании событий - пользователь будет оповещен об этом за 2 секунды и если уведомление будет мешать пользователю, то он может кликнуть на неё и уведомление исчезнет / иначе нет;
	- (превью: "динамическое"/"статическое"), то при каждом нажатии на клавишу будет создаваться контент - динамически / иначе при потери фокуса поля;
	- (функциональность.очистка: "да"/"нет") - появиться кнопка, при клике на которую поля ввода будут очищены / иначе кнопка будет удалена;
	- (функциональность.копирование: "да"/"нет") - появиться кнопка, при клике на которую - записи из полей ввода будут скопированы в буфер обмена данными / иначе кнопка будет удалена;
	- (функциональность.сохранение: "да"/"нет") - появиться кнопка, при клике на которую - записи из полей ввода будут сохранены в локальное хранилище браузера и при повторном входе будет предложено загрузка данных / иначе кнопка будет удалена;
По умолчанию:
	- Кнопки обеспечивающие функциональность сайта скрыты;
	- Всё включено;
